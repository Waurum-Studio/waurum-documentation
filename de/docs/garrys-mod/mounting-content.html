🟪 Mounting CSS content

Having map errors / missing textures?
Follow bellow steps to get that sorted.

Without Counter-Strike Source​

info

Please, skip to the next section if you already own the game Counter-Strike Source.

Requirements​

  • Garry's Mod must be installed already
  • WinRAR or any software that can extract ZIP / RAR archives

Video Version​

https://youtu.be/8AfuNpX8RQA?t=154

Text Version​

  1. Find a convenient website and download the CSS content

    We recommend using this one

  2. Extract the content into a new folder named csscontent for example
You should have a similar file tree: 

csscontent
├── materials
├── models
├── particles
└── sound
  1. Find your Garry's mod addons folder on your computer

    Common path is C:\Program Files (x86)\Steam\steamapps\common\GarrysMod\garrysmod\addons

Folder location

Note that the path can vary based on where your games and Steam are installed on your computer.

  1. Move the csscontent folder into your addons folder
You must have the following file tree:

addons
└── csscontent (or whatever you named it)
    ├── materials
    ├── models
    ├── particles
    └── sound
  1. Restart or start your game and issue should be fixed!

With Counter-Strike Source​

info

Garry's mod should automatically mount CSS content if it detects that Counter-Strike Source is installed.
The following steps can be used instead to manually mount it.

Requirements​

  • Garry's Mod and Counter-Strike Source should be installed
  • Garry's Mod must be able to access Counter-Strike Source folder at any time

Video Version​

https://youtu.be/8AfuNpX8RQA?t=130

Text Version​

  1. Start Garry's Mod
  2. In the main menu, go to the bottom right and hit Games button

    It should have a 🎮 icon

  3. Verify that Counter-Strike source is ✅, if not check it
  4. CSS content are now mounted!
